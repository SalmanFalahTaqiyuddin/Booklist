## ğŸš€ Instalasi & Setup Proyek

Ikuti langkah-langkah sederhana di bawah ini untuk menjalankan proyek Booklist di perangkat lokal Anda.

### Prasyarat

Pastikan Anda telah menginstal komponen berikut di sistem Anda:

1.  **Flutter SDK** (Versi 3.x atau terbaru)
2.  **Dart SDK**
3.  IDE (VS Code atau Android Studio) dengan *plugin* Flutter/Dart terinstal.

### Langkah-langkah Instalasi

1.  **Clone Repositori:**
    Buka terminal dan *clone* repositori proyek ini. Ganti `https://github.com/SalmanFalahTaqiyuddin/Booklist` dengan URL repositori Anda yang sebenarnya.
    ```bash
    git clone https://github.com/SalmanFalahTaqiyuddin/Booklist
    cd booklist-app
    ```

    2.  **Ambil Dependensi:**
    Jalankan perintah berikut untuk mengunduh semua *package* (dependensi) yang dibutuhkan oleh proyek:
    ```bash
    flutter pub get
    ```

3.  **Jalankan Aplikasi:**
    Pastikan Anda memiliki emulator (Android/iOS) atau perangkat fisik yang terhubung. Kemudian jalankan aplikasi:
    ```bash
    flutter run
    ```
